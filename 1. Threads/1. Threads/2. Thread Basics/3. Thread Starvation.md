## Thread Starvation

### Definition

Thread starvation occurs when a **thread is unable to gain regular access to shared resources**, so it **cannot make progress** and remains stuck for a long time.

In simple words:

> A thread is *ready to work* but **never gets the chance** to execute.

---

### Why Thread Starvation Happens

Thread starvation usually occurs due to:

* **Too many active threads** competing for limited resources
* **Unfair scheduling policies** (e.g., priority-based scheduling where high-priority threads dominate)
