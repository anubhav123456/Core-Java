# ✅ **Lambda Expressions in Java**

---

# **1. What is a Lambda Expression?**

* Introduced in **Java 8**
* A **lambda expression is an anonymous function**
  → It has **no name**, **no access modifiers**, and **no return type mentioned**
* It is used to provide implementation of **Functional Interfaces**
  (interfaces with **Single Abstract Method – SAM**)

---

# **2. Lambda vs Traditional Method**

| Feature       | Java Method     | Lambda Expression              |
| ------------- | --------------- | ------------------------------ |
| Name          | Must have       | No name (anonymous)            |
| Class Binding | Part of a class | No class required              |
| Return Type   | Must mention    | Inferred automatically         |
| Parameters    | Required        | Required                       |
| Usage         | General purpose | Only for Functional Interfaces |

---

# **3. Syntax of Lambda Expression**

```
(parameters) -> { body }
```

Components:

* **()** → parameters
* **->** → arrow
* **{}** → function body

---

# **4. Functional Interfaces**

A functional interface has exactly **one abstract method**.

Examples in Java:

| Interface  | Method    |
| ---------- | --------- |
| Runnable   | `run()`     |
| Callable   | `call()`    |
| Comparator | `compare()` |

# ✅ **A. Lambda Without Parameters**

# ----------------------------------------------

## **Functional Interface**

```java
@FunctionalInterface
public interface HelloWorldInterface {
    String sayHelloWorld();
}
```

## **Traditional Implementation**

```java
public class HelloWorldTraditional implements HelloWorldInterface {
    @Override
    public String sayHelloWorld() {
        return "Hello World!";
    }
}
```

## **Lambda Implementation**

```java
public class Main {
    public static void main(String[] args) {

        HelloWorldTraditional traditional = new HelloWorldTraditional();
        System.out.println(traditional.sayHelloWorld());

        HelloWorldInterface helloWorldInterface1 = () -> {
            return "Hello World Using Lambda!";
        };

        HelloWorldInterface helloWorldInterface2 = () -> 
                "Hello World Using Lambda but without body!";

        System.out.println(helloWorldInterface1.sayHelloWorld());
        System.out.println(helloWorldInterface2.sayHelloWorld());
    }
}
```

### **OUTPUT**

```
Hello World!
Hello World Using Lambda!
Hello World Using Lambda but without body!
```

# ✅ **B. Lambda With One Parameter**

# ----------------------------------------------

## **Functional Interface**

```java
@FunctionalInterface
public interface IncrementByFiveInterface {
    int incrementByFive(int x);
}
```

## **Traditional**

```java
public class IncrementByFiveTraditional implements IncrementByFiveInterface {
    @Override
    public int incrementByFive(int x) {
        return x + 5;
    }
}
```

## **Lambda Implementation**

```java
public class Main {
    public static void main(String[] args) {

        IncrementByFiveTraditional traditional = new IncrementByFiveTraditional();
        System.out.println(traditional.incrementByFive(6));

        IncrementByFiveInterface interface1 = (x) -> {
            return x + 5;
        };

        IncrementByFiveInterface interface2 = (x) -> x + 5;

        System.out.println(interface1.incrementByFive(6));
        System.out.println(interface2.incrementByFive(6));
    }
}
```

### **OUTPUT**

```
11
11
11
```

# ✅ **D. Lambda Using Runnable**

# ----------------------------------------------

```java
public class Main {
    public static void main(String[] args) {

        Runnable runnable1 = new Runnable() {
            @Override
            public void run() {
                int sum = 0;
                for(int i = 1; i <= 10; i++) sum += i;

                System.out.println("The sum is " + sum);
            }
        };

        Runnable runnable2 = () -> {
            int sum = 0;
            for(int i = 1; i <= 10; i++) sum += i;

            System.out.println("The sum is " + sum);
        };

        Thread thread1 = new Thread(runnable1);
        Thread thread2 = new Thread(runnable2);

        Thread thread3 = new Thread(() -> {
            int sum = 0;
            for(int i = 1; i <= 10; i++) sum += i;

            System.out.println("The sum is " + sum);
        });

        thread1.start();
        thread2.start();
        thread3.start();
    }
}
```

### **OUTPUT**

(All three output the same, order may vary)

```
The sum is 55
The sum is 55
The sum is 55
```

# ✅ **E. Lambda Using Callable**

# ----------------------------------------------

```java
import java.util.concurrent.*;

public class Main {
    public static void main(String[] args) {

        Callable callable1 = new Callable() {
            @Override
            public Object call() throws Exception {
                int sum = 0;
                for(int i = 1; i <= 10; i++) sum += i;
                return sum;
            }
        };

        Callable callable2 = () -> {
            int sum = 0;
            for(int i = 1; i <= 10; i++) sum += i;
            return sum;
        };

        ExecutorService executor = Executors.newFixedThreadPool(3);

        Future future1 = executor.submit(callable1);
        Future future2 = executor.submit(callable2);
        Future future3 = executor.submit(() -> {
            int sum = 0;
            for(int i = 1; i <= 10; i++) sum += i;
            return sum;
        });

        try {
            System.out.println("Output from Callable 1 = " + future1.get());
            System.out.println("Output from Callable 2 = " + future2.get());
            System.out.println("Output from Callable 3 = " + future3.get());
        }
        catch (Exception e) {
            e.printStackTrace();
        }

        executor.shutdown();
    }
}
```

### **OUTPUT**

```
Output from Callable 1 = 55
Output from Callable 2 = 55
Output from Callable 3 = 55
```
