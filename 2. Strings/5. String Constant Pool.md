## âœ… **String Constant Pool**
* The String Constant Pool (SCP) is a special memory area inside the heap where Java stores string literals. 
* It helps in saving memory by reusing immutable String objects instead of creating duplicates.
```java
String s1 = new String("Hello");
String s2 = "Hello";
String s3 = "Hello";
String s4 = s1.intern();
System.out.println(s1 == s2); // false
System.out.println(s2 == s3); // true
System.out.println(s2 == s4); // true
```

## âœ… **String Immutability & Object Creation (Heap vs SCP)**
```java
String s = new String("Durga");
s.concat("software");
s = s.concat("solutions");
System.out.println(s);
```
| Type | Objects Created | Explanation |
|------|----------------|-------------|
| Heap | 3 | `"Durga"` (from `new`), `"Durgasoftware"` (unused), `"Durgasolutions"` (assigned to `s`) |
| SCP (String Constant Pool) | 3 | `"Durga"`, `"software"`, `"solutions"` |

```java
String s1 = new String("Spring");
s1.concat("Fall");
String s2 = s1.concat("Winter");
s2.concat("Summer");
System.out.println(s1);
System.out.println(s2);
```
| Memory Type | Objects Created | Details |
|-------------|----------------|---------|
| Heap | 4 | `"Spring"`, `"SpringFall"`, `"SpringWinter"`, `"SpringWinterSummer"` |
| SCP (String Constant Pool) | 4 | `"Spring"`, `"Fall"`, `"Winter"`, `"Summer"` |
