## âœ… **charAt(int index)**
* Returns the **character** present at a specific index (position) in the string.
* Index starts from 0.
```java
String s = "Java";
System.out.println(s.charAt(1)); // Output: 'a'
```
---

## âœ… **isEmpty()**
* Checks if the string is **empty** (i.e., has length 0).
* Returns **true** if empty, **false** otherwise.
```java
String s1 = "";
System.out.println(s1.isEmpty()); // true
```
---

## âœ… **length()**
* Returns the number of characters in the string.
```java
String s = "Hello";
System.out.println(s.length()); // 5
```
---

## âœ… **replace(char oldChar, char newChar)**
* Replaces all occurrences of one character with another.
```java
String s = "banana";
System.out.println(s.replace('a', 'o')); // "bonono"
```
* You can also replace substrings.
```java
String s2 = "Java Programming";
System.out.println(s2.replace("Java", "Python")); // "Python Programming"
```
---

## âœ… **substring(int beginIndex)**
* Returns a part (substring) of the string starting from beginIndex (inclusive) till end of string.
```java
String s = "JavaWorld";
System.out.println(s.substring(4)); // "World"
```
---

## âœ… **substring(int beginIndex, int endIndex)**
* Returns a part (substring) of the string starting from beginIndex (inclusive) to endIndex (exclusive).
```java
String s = "JavaWorld";
System.out.println(s.substring(0, 4)); // "Java"
```
---

## âœ… **indexOf(String str)**
* Returns the **index (position)** of the **first occurrence** of a character or substring.
* Returns `-1` if not found.
```java
String s = "programming";
System.out.println(s.indexOf('g')); // 3
System.out.println(s.indexOf("ram")); // 4
```
---

## âœ… **indexOf(String str, int fromIndex)**
* Returns the **index (position)** of the **first occurrence** of a character or substring, but start looking from fromIndex.
* Returns `-1` if not found.
```java
String s = "geeksforgeeks";
System.out.println(s.indexOf("geeks", 1)); // 8
```
---


## âœ… **lastIndexOf(String str)**
* Returns the index of the last occurrence of a character or substring.
```java
String s = "programming";
System.out.println(s.lastIndexOf('g')); // 10
```
```java
String s = "geeksforgeeks";
System.out.println(s.lastIndexOf("geeks"));
```
---


## âœ… **toLowerCase()**
* Converts all characters in the string to lowercase.
```java
String s = "HELLO";
System.out.println(s.toLowerCase()); // "hello"
```
---


## âœ… **toUpperCase()**
* Converts all characters in the string to uppercase.
```java
String s = "hello";
System.out.println(s.toUpperCase()); // "HELLO"
```
---

## âœ… **trim()**
* Removes leading and trailing spaces (but not spaces in between).
```java
String s = "   Java  ";
System.out.println(s.trim()); // "Java"
```
