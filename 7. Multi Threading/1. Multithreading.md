# Motivation for Multithreading

---

## 1. Why Multithreading Is Needed

### Default Nature of Programs

Most programming languages execute code **sequentially**, i.e., **line by line**.

```java
public class Runner {
    public static void main(String[] args) {
        initDB();
        downloadData();
        processOnData();
        showTheResults();
    }
}
```

### What Happens Here?

* `initDB()` runs first
* then `downloadData()`
* then `processOnData()`
* finally `showTheResults()`

Each step **waits** for the previous one to finish.

## 4. Formal Definition of Multithreading

> **Multithreading** is the ability of a CPU/system to execute **multiple tasks concurrently** by using multiple threads within a process.
